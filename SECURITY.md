# Security Policy

## Supported Versions

Only the latest version is supported. We don't do LTS releases.
Breaking changes are reported in the [changelog/history](history.md) file.

| Version | Supported          |
|---------|--------------------|
| 0.5.x   | :white_check_mark: |
| 0.4.x   | :x:                |
| 0.3.x   | :x:                |
| 0.2.x   | :x:                |
| < 0.1   | :x:                |
