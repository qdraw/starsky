    • starskyImporterCli import command line interface
    • starskyGeoCli gpx sync and reverse 'geo tagging'

    • starskyWebFtpCli copy a content package to a ftp service

  • starsky.netframework Client for older machines
  • starsky-tools nodejs tools to add-on tasks
  • starskyapp Desktop Application (Pre-alpha code)

Build instructions

The following steps are needed to compile the desktop application

Install Node packages

npm ci

Building Starsky as dependency first

For Windows

When running bash

../starsky/build.sh --Runtime=win7-x86

For Mac OS

When running bash

../starsky/build.sh --Runtime=osx.10.12-x64

Build Electron

For Windows

When running bash

npm run windows

For Mac OS

When running bash

npm run mac
